# Test Corpus

This directory contains the dataset for verifying the RAG pipeline.

## Contents
1. `papers/`: Place 5 open-access PDFs here.
2. `dummy_citing_paper.pdf`: A document that contains citations `[1]` through `[5]` matching the papers in the directory.

## Testing Forward Mode
Paste sentences from any of the ingested papers into the search bar.

## Testing Backward Mode
Open `dummy_citing_paper.pdf` in the viewer and hover over the `[N]` citations.
